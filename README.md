### Web Technologies (WT)
- Languages used in WT are `HTML` `CSS` `JavaScript` `Java` `JDBC` `Java Servlets` and `Java Server Pages (JSP)`
- The code for every week is given in individual folder with name convention `WT_WEEK_0`
